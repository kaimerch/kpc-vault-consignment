# KPC Vault Consignment - Quick Reference Guide

## 🎯 APP FEATURES

### Commission Calculator
- **High-value items** ($500+): 25% commission
- **Standard items** ($100-$499): 33% commission  
- **Low-value items** (<$100): 40% commission
- **Specialty items**: 35% commission
- Real-time calculation with breakdown

### Client Intake Process
1. Client fills out personal information
2. Adds items with photos and descriptions
3. Reviews commission estimates
4. Signs digital contract
5. Receives confirmation

### Client Portal
- Login with client ID
- View all consigned items
- Track item status (pending/active/sold)
- View sales history and earnings
- Download contracts

## 📱 CLIENT WORKFLOW

### For New Clients:
1. Send to: `your-app.vercel.app/consign`
2. They complete intake form
3. System creates Airtable records
4. Generates PDF contract
5. Client receives confirmation

### For Existing Clients:
1. Send to: `your-app.vercel.app/portal`
2. They enter their client ID
3. View all items and earnings
4. Track sales progress

## 💼 BUSINESS MANAGEMENT

### Airtable Dashboard
- **Clients table**: All customer info and contact details
- **Items table**: Complete inventory with photos and status
- **Contracts table**: All signed agreements and PDFs
- **Sales table**: Transaction history and payouts

### Daily Operations
1. Check new intake submissions in Airtable
2. Update item statuses as they sell
3. Record sales in Sales table
4. System calculates payouts automatically
